# Todo List App (React Native + Expo)

Esta es una aplicación simple de lista de tareas (To-Do List) desarrollada con React Native, Expo y TypeScript. Permite a los usuarios agregar tareas, marcarlas como completadas y eliminarlas.

## 📋 Características

- Agregar nuevas tareas.
- Marcar tareas como completadas (tachado y check).
- Eliminar tareas de la lista.
- Interfaz limpia y funcional.

## 🚀 Requisitos Previos

Antes de comenzar, asegúrate de tener instalado lo siguiente:

1.  **Node.js**: [Descargar Node.js](https://nodejs.org/)
2.  **Expo Go (En tu celular)**: Descárgala desde la App Store (iOS) o Google Play (Android).

## 🛠️ Instalación y Configuración

Sigue estos pasos para descargar y ejecutar el proyecto localmente:

1.  **Clonar el repositorio:**
    ```bash
    git clone <AQUI_IRÁ_TU_URL_DEL_REPOSITORIO>
    ```

2.  **Navegar al directorio del proyecto:**
    ```bash
    cd my-todo-list
    ```

3.  **Instalar dependencias:**
    ```bash
    npm install
    # o si usas yarn:
    yarn install
    ```

## 📱 Ejecución

Para iniciar la aplicación en modo desarrollo:

1.  Ejecuta el siguiente comando en la terminal:
    ```bash
    npx expo start
    ```

2.  Se mostrará un código QR en la terminal.
3.  Abre la aplicación **Expo Go** en tu dispositivo Android o usa la cámara en iOS para escanear el código QR.
4.  ¡Listo! La aplicación cargará en tu teléfono.

## 📂 Estructura del Proyecto

- `app/index.tsx`: Contiene toda la lógica y la interfaz de usuario de la aplicación.
- `package.json`: Lista de dependencias y scripts del proyecto.
